Overview:

The Domain Change Predictor (DoChaP) integrates information on exons and domains from several sources to allow the study of the effect of alternative splicing on protein domains.  

Search DoChaP:

  • Choose a species from the list: Human (hg38), Mouse (mm10), Rat (rn6), Zebrafish (danRer11), Frog (xenTro9). To request another species, contact us! galozs@post.bgu.ac.il
  • Enter a search term:
    1. Acceptable query formats: Gene, Transcript or protein identifiers (gene symbol/ NCBI’s Gene ID or RefSeq transcript or protein ID / ENSEMBL gene, transcript or protein ID)
    2. See examples or Help page for more info
  • Ignore predicted genes (shown by default) – check box to show or hide predictions
