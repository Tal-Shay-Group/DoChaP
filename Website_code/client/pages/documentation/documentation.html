How to use DoChaP?

On the homepage of DoChaP you will find the search window where you need to insert your search term and choose a species.

A - Species: 

Human (hg38) or Mouse (mm10). More species will be added in the future. We invite users to contact us to request specific species of interest.

B - Search by gene, transcript or protein:

In the search box, enter one of the following gene identifiers: gene symbol, gene synonym, NCBIâ€™s Gene ID, RefSeq transcript or protein ID (starts with NM_ /XM_/NP_/XP_), ENSEMBL gene, transcript or protein ID (starts with ENS[species prefix]G / ENS[species prefix]T / ENS[species prefix]P)

C - Ignore predicted records:

Predicted records are transcripts that have not yet been validated by the providers of the source information; for example: transcripts or proteins with a RefSeq ID that starts with XM_ or XP_ (respectively). By checking this check box, you will remove all the predictions from your results.

Compare species transcripts with DoChaP:

The DoChaP "Compare Species" feature enables the user to visualize and compare transcripts from different species. To compare transcripts, choose the species you wish to compare and a gene of interest in a similar format to the DoChaP search for a single species, as explained above.

The results will be all the transcripts of the gene in species A followed by all the transcripts of the gene in species B, shown in the same format as described above for DoChaP main page.

 

What are the sources of the information in DoChaP?

DoChaP integrates data from several sources, all downloaded from the relevant ftp sites:

  • Exon annotation â€“ parsed from the ncbiRefSeq table downloaded from the UCSC database for human (Version) and mouse (version).
  • Protein domain annotation â€“ parsed from GeneBank .gpff files from the NCBI  RefSeq  database
  • Other tables used for accession ID connections are: kgXref and knownGene tables from the UCSC database and the gene2ensemble table from the NCBI Gene database

All information was parsed and organized into a single database for the use of DoChaP webserver.

Note:

  • Only protein coding genes are shown in DoChaP
  • Domains are only the "Regions" features from NCBI RefSeq fulfilling the following criteria:
    • Records with accession ID of known domain databases (NCBI Curated Domains, Pfam, SMART, COG, PRK, TIGRFAMs)
    • Records larger than 1 amino acid
