The Domain Change Presenter (DoChaP) is a web server for visualization of exon–domain relationships in different transcripts. DoChaP provides an easy way of exploring the effect of alternative splicing on protein domains.

Alternative RNA splicing results in multiple transcripts of the same gene, possibly coding different protein isoforms that may differ in their protein domains. Although the information of transcripts and domains can be extracted from public databases, such a process requires the integration of data from several sources and it is tedious and unfeasible for genome-wide analysis. The DoChaP visualization integrates information on transcripts and domains to facilitate the study of the effect of alternative splicing on domain structure. Identifying such an effect can lead to testable hypotheses in the form of 'protein X loses domain Y in state Z'. 

Sources of information:

DoChaP integrates data from NCBI’s RefSeq, Ensembl and Interpro into a single combined database. Data sources:

  • Genomic annotations (genes, exons, CDS, transcription) – .gff files downloaded from NCBI's genomes - RefSeq and Ensembl ftp sites (the latest version available at the time of DoChaP update)
  • Protein domain annotation – .gpff files were downloaded from the NCBI RefSeq database and Ensembl records were downloaded from BioMart. Those sources contain multiple domain prediction databases. DoChaP displays: NCBI Curated Domains, Pfam, SMART, TIGRFAMs, InterPro
  • Orthology data – was downloaded from Ensembl using BioMart.
  • Other tables used for accession ID connections are: gene2ensemble table from the NCBI Gene database and InterPro entries table downloaded from InterPro.

All information was parsed and organized into a single database for presentation on the DoChaP webserver.

Notes:

  • Only protein coding transcripts are shown in DoChaP
  • Transcripts encoded from alternate locus to the reference genomes are not included in DoChaP database
